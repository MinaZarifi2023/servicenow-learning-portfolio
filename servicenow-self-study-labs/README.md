# ServiceNow Self-Study Labs

This repository documents my self-study journey into the ServiceNow platform, with a focus on ITSM fundamentals, basic administration concepts, workflow automation, and introductory scripting.

> Note: This is a learning portfolio project based on Now Learning concepts and a personal developer instance. It is not a production ServiceNow implementation.

## Goals

- Understand the basic ServiceNow platform structure and navigation
- Learn ITSM flows such as Incident, Change, and Request Management
- Practice Service Catalog and Flow Designer concepts
- Understand Users, Groups, Roles, and ACL fundamentals
- Build first knowledge of JavaScript Business Rules and Client Scripts
- Connect AI/ML background with ServiceNow use cases such as ticket classification and workflow automation
