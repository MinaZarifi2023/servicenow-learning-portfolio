# ServiceNow Concepts

## Instance

A ServiceNow instance is a separate environment of the ServiceNow platform. Companies often have development, test, and production instances.

## Table

ServiceNow stores data in tables. For example, incidents are stored in an incident table.

## Record

A record is one row in a table. One incident ticket is one record.

## List

A list shows multiple records from a table.

## Form

A form shows the details of one record.

## Incident

An incident is an unplanned interruption or reduction in quality of an IT service.

## Request

A request is a standard user demand, such as requesting software, hardware, or access.

## Change

A change is a planned modification to an IT service or system.

## Service Catalog

The Service Catalog allows users to request predefined services.

## Flow Designer

Flow Designer is a low-code tool for automating processes.

## ACL

Access Control Lists define who can access data and what actions they can perform.
