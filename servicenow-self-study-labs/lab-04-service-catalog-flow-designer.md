# Lab 04 – Service Catalog and Flow Designer

## Objective

Understand how ServiceNow can be used to automate standard requests.

## Concepts

- Service Catalog
- Catalog Item
- Variables
- Approval
- Flow Designer
- Trigger
- Action
- Condition

## Example Scenario

An employee requests a new laptop through the Service Catalog. The request needs manager approval and then goes to the IT hardware team.

## Designed Workflow

1. User submits catalog item: New Laptop Request
2. System creates request record
3. Manager approval is requested
4. If approved, task is assigned to IT hardware group
5. User receives notification
6. Request is closed after fulfillment

## Reflection

Service Catalog and Flow Designer are powerful tools for standardizing and automating repeatable business processes.
